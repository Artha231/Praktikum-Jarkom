  # laporan praktikum Jaringan komputer IF-04-04

  ## Tujuan Praktikum

  mempelajari  wireshark 

  ## Langkah percobaan 
  1. install iso wireshark 
  2. instalasi wireshark
  3. konfigurasi wireshark ke http://gaia.cs.umass.edu/wireshark-labs/INTRO-wireshark-file1.html 
  4. hapus https jadi http
  5. filter wireshark nya biar bisa di akses  website http://gaia.cs.umass.edu/wireshark-labs/INTRO-wireshark-file1.html dengan cara ketik http
  6. jika udah keliatan tulisan Congratulations! You've downloaded the first Wireshark lab file! di website dan ada tampilan http saja di wireshark
